# Assets Folder

Place your campus images here for use in the application.

## Required Image

**File:** `astu-gate.jpg`  
**Location:** `public/assets/astu-gate.jpg`  
**Description:** The ASTU Gada Gate entrance image showing the main campus gate with palm trees and modern architecture.

### How to Add the Image

1. Save your campus gate image as `astu-gate.jpg`
2. Place it in this folder (`public/assets/`)
3. The landing page will automatically display it in the hero section with modern styling

### Image Specifications

- **Recommended dimensions:** 1920x1080px or higher
- **Format:** JPG, PNG, or WebP
- **Aspect ratio:** 16:9 or similar landscape orientation
- **File size:** Optimized for web (under 500KB recommended)
